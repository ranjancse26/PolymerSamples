# PolymerSamples

Polymer.js based web component sample projects

## Setup

1) Install Bower 

npm install -g bower

2) Install Polymer CLI

npm install -g polymer-cli

# Contact Form

Please run the below command to run the App

polymer serve --open

or

polymer serve --port 8081 (You can specify any port)
